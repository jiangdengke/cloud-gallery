# 任务清单

## 基础搭建
- [ ] 定义路由与页面结构（浏览页、管理页）。
- [ ] 设计 API 结构：目录列表、文件详情、预览、上传、管理操作。
- [ ] 明确存储根路径与公开访问路径（`storage/app` + `storage:link`）。

## 权限与会话
- [ ] 新增 Key 校验接口（基于环境变量）。
- [ ] 登录后创建会话（Session/Cookie），记录管理权限。
- [ ] 前端上传按钮在未验证时提示输入 Key。

## 目录浏览
- [ ] 目录列表接口：支持路径参数与基础排序。
- [ ] 面包屑导航与返回上级。
- [ ] 文件详情与下载链接。

## 预览能力
- [ ] 图片预览（常见格式：jpg/png/webp/gif）。
- [ ] 视频播放（常见格式：mp4/webm）。
- [ ] 其他文件显示为可下载链接。

## 上传与管理
- [ ] 上传接口（单文件/多文件）。
- [ ] 创建目录、重命名、删除操作。
- [ ] 操作确认与错误提示。

## 体验与稳定性
- [ ] 统一错误响应与前端提示。
- [ ] 基础日志与异常处理。
- [ ] 可选：基础测试（上传、浏览、权限）。

## 验收标准
- [ ] 访问首页可浏览目录结构。
- [ ] 预览图片与视频正常可用。
- [ ] 未输入 Key 只能浏览，输入 Key 后可上传与管理。
- [ ] 上传成功后可在目录中看到并访问文件。
